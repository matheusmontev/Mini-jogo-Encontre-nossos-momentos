leia a pasta `inspiração/` antes de qualquer alteração visual

Toda vez que for modificar, criar ou revisar qualquer tela, componente ou estilo deste projeto, você deve ler primeiro:

1. `inspiração/velvet_memory/DESIGN.md`  
   → Contém a paleta de cores oficial, tipografia, espaçamentos, bordas, elevação e diretrizes de estilo do projeto. Nenhuma cor, sombra ou fonte deve ser inventada — use apenas o que está definido aqui.

2. Qualquer arquivo png dentro de `inspiração/`  
   → São as telas de referência visual. O layout, hierarquia de componentes e estrutura de cada tela devem seguir esses arquivos como modelo.
