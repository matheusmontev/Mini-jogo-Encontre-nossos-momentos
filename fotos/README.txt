# Como Adicionar Fotos

1. Coloque suas fotos nesta pasta (fotos/).
2. Recomendamos renomear as fotos para nomes simples como "memoria1a.jpg", "memoria1b.jpg".
3. Certifique-se de que cada memória possui duas fotos (para fazerem par no jogo).
4. Edite o arquivo "../data/memorias.js" para atualizar os itens do jogo referenciando os nomes das fotos que você inseriu aqui.
